EFEITO PARALAX: CENTRALIZANDO O TÍTULO

Centralizando um título no background

No html

div class="caixa1"
div class="conteudo-titulo"
span class="titulo">CONTEÚDO 1 /span

No CSS

.conteudo-titulo{
position: absolute;
left:0; /*é alinhado a partir da borda esquerda então não fica bem centralizado...Qto maior o texto, menos centralizado*/
top:45%;
width: 100%;
text-align: center; /*aqui ele é alinhado centralizado. não da pra por left:50% pois ele alinha a partir da margem esquerda do elemento. se o conteúdo for grande, não ficará alinhado*/
}

.conteudo-titulo span.titulo{
color: #fff;
background: #111;
padding: 18px;
font-size: 20px;
letter-spacing: 5px;
}

EFEITO PARALAX

PARALAX

É um efeito conseguido utilizando camadas de imagens.
Para conseguir o efeito paralax, temos que ter no css BODY e HTML com height: 100%.

Quando colocamos uma borda, vemos que com height 100%, ele ocupa 100% do tamanho da tela do usuário. É o que queremos para ter o efeito paralax. Se não escrevêssemos isso, o body ocuparia o tamanho do conteúdo da página. Criamos 3 divs no exemplo do professor. A div 1 tem que ter também o height (altura) de 100%. ENTÃO O BODY, HTML E A IMAGEM QUE VAI FICAR POR BAIXO PARADA ENQUANTO SUBIMOS A TELA, TEM QUE ESTAR 100%. Vamos ter uma formatação padrão para as divs com as imagens no background. As divs com conteúdo chamei de class=”conteúdo” e não participa dessa formação para o efeito paralax. As divs com imagens DEVEM TER HEIGHT 100%.

A formatação padrão das divs que contém imagem que vai ficar no fundo sendo sobreposta, dando o efeito paralax é:
.caixa1, .caixa2, .caixa3, .caixa4 {
border: 1px solid red; (aqui é só para visualizarmos que ocupa toda a tela. depois podemos retirar a borda
height: 100%;
opacity: 0.7;
background-repeat: no-repeat;
background-size: cover;
background-position: center;
background-attachment: fixed;
}

IMAGEM PARA BACKGROUND DO PARALAX

A ideia é ter uma imagem que seja grande o suficiente para cobrir a maior parte dos dispositivos. Background-repeat: no-repeat impede que a imagem fique se repetindo. Background-size:cover faz com que a imagem cubra toda a tela. E background-position: center faz com que a imagem esteja sempre centralizada, mesmo quando reduzimos a tela. Background-attachment: fixed é o que faz com que a imagem fique fixa e as demais passem sobre ela. É o que dá o efeito paralax.
